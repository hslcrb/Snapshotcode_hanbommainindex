# 한봄고등학교 홈페이지 스냅샷 (Hanbom High School Website Snapshot)

## 통합 웹 구조 (Unified Project Structure)
표준 웹 프로젝트 규격에 맞춰 디렉토리 구조가 다음과 같이 통합되었습니다:
The project has been unified into a standard web structure:

- **`index.html`**: 메인 진입 페이지 (Main entrance page)
- **`assets/`**: 모든 에셋 통합 관리 (Unified assets management)
  - `css/`: 스타일시트 (Stylesheets)
  - `js/`: 자바스크립트 및 인터랙티브 스크립트 (JavaScript & interactive scripts)
  - `images/`: 이미지 및 로고 에셋 (Images & logo assets)
  - `fonts/`: 로컬 및 외부 웹 폰트 (Local & web fonts)
  - `uploads/`: 게시판 등 미디어 업로드 파일 (Media upload files)
- **`docs/`**: 상세 분석 및 구조 문서 (Detailed analysis and structure docs)
- **`backups/`**: 원본 소스 데이터 (Original source data)
